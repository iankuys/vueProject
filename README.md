# Welcome to my vue js Inventory Management system
## Name: Ku Yi Sien
## Please follow the instruction steps below to run the program

## Project setup
```
npm install
```
## Install LocalBase (api that uses indexeddb)
```
npm install localbase --save
```

npm run serve
